# Finanzas App

Aplicación de gestión financiera personal para el seguimiento de ingresos, gastos y transferencias entre cuentas.

## Objetivo
Facilitar el control de las finanzas personales mediante una interfaz intuitiva, permitiendo al usuario visualizar su estado financiero en tiempo real y gestionar sus movimientos de forma eficiente.

## Características Principales
- Gestión de Cuentas (Efectivo, Banco, Billetera).
- Categorización de Ingresos y Gastos.
- Registro de Transferencias entre cuentas.
- Dashboard con resumen mensual, saldos actuales y acceso rápido a transacciones por cuenta.
- Filtros avanzados por fecha, categoría y cuenta.
- Interfaz optimizada para móviles con diálogos de confirmación y diseño responsivo.

## Stack Tecnológico
- **Frontend:** Next.js 15+ (App Router), Tailwind CSS, Lucide React, Framer Motion.
- **Backend/Base de Datos:** Firebase Firestore.
- **Autenticación:** Firebase Auth (Google Login).
- **Notificaciones:** Sonner.
- **Gestión de Estado:** React Hooks y Context API.
